Markup project

Structure based on Html5Boilerplate.

Styles powered with Sass and Compass.

There are images for IE8 and IE7 in /img/base64. For all modern browsers they are converted to base64 images.

IE compatibility.
version 9 - full suport
version 8 - graceful degradation
version 7 - more graceful degradation
version 6 - unsupported
